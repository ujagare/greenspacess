# Google Search Console Verification - Quick Guide

## Method 1: Meta Tag (EASIEST - Recommended)

### Steps:
1. Go to Google Search Console
2. Click "Verify" 
3. Choose "HTML tag" method
4. Copy the meta tag (looks like this):
   ```html
   <meta name="google-site-verification" content="YOUR_CODE_HERE" />
   ```
5. Send me that meta tag
6. I'll add it to all pages
7. Click "Verify" in Search Console

---

## Method 2: HTML File Upload

### Steps:
1. Go to Google Search Console
2. Choose "HTML file" method
3. Download the verification file (e.g., google1234567890abcdef.html)
4. Send me the filename
5. I'll create it in root folder
6. Upload to your website root
7. Click "Verify" in Search Console

---

## Which Method to Use?

**Use Meta Tag Method** - Easier, no file upload needed
**Use HTML File** - If meta tag doesn't work
