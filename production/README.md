# Greenspaces - Premium Land Projects

A professional website for showcasing premium agriculture land and farmhouse plots in Pune region.

## Features
- Responsive design
- Project portfolio (Purandhar, Khed, Kokan, Mulshi)
- Contact forms
- Interactive animations

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery

## How to Run
1. Clone the repository
2. Open `index.html` in your browser

## Projects
- **Purandhar Project** - Premium Agriculture Land
- **Khed Project** - Lake Touch Farmhouse Plots
- **Kokan Project** - Scenic Valley Plots
- **Mulshi Project** - Premium NA Plots
